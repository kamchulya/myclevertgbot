# САК Планер — Telegram Bot

Личный планер с утренними вопросами, вечерними напоминаниями,
голосовым вводом и дневником мыслей. Деплоится на Railway за 5 минут.

---

## Структура репозитория

```
sak-planner-bot/
├── bot.py           ← главный файл бота
├── web.py           ← веб-сервер для планера (Mini App)
├── static/
│   └── index.html   ← HTML-планер (открывается кнопкой в боте)
├── requirements.txt ← зависимости Python
├── Procfile         ← инструкция для Railway
├── railway.json     ← конфиг Railway
└── README.md
```

---

## Деплой на Railway — 5 шагов

### 1. Создай бота
Открой @BotFather → `/newbot` → скопируй токен

### 2. Узнай свой Telegram ID
Напиши @userinfobot — он ответит твоим `id`

### 3. Залей на GitHub
Создай репозиторий, загрузи все файлы

### 4. Деплой на Railway
- Зайди на railway.app
- New Project → Deploy from GitHub repo
- Выбери репозиторий

### 5. Добавь переменные (Variables)

| Переменная | Значение |
|-----------|---------|
| `BOT_TOKEN` | токен от @BotFather |
| `OWNER_ID` | твой Telegram ID |
| `OPENAI_API_KEY` | ключ OpenAI (для голоса) |
| `TZ` | `Asia/Almaty` |
| `MORNING_H` | `7` |
| `EVENING_H` | `20` |

После добавления переменных — Railway сам перезапустит бота.

## Что умеет бот

| Команда / кнопка | Действие |
|-----------------|---------|
| `/start` | Запуск |
| 📋 Задачи | Список задач на сегодня |
| ➕ Добавить задачу | Добавить текстом или голосом |
| 💭 Мысль в дневник | Записать мысль (текст или голос) |
| ⚖️ Записать вес | Вес сегодня |
| 📊 Статистика | Прогресс и динамика |
| 🗂 Открыть планер | Mini App внутри Telegram |
| Утром 07:00 | Бот спрашивает задачи на день |
| Вечером 20:00 | Бот напоминает отметить задачи |
